An agent is a model in a loop with tools and a goal. Most of the difficulty
is not the model — it's the harness:

- **Tool design**: small, composable, well-described tools beat clever ones
- **Context management**: what the agent sees *is* the agent
- **Verification**: agents that can check their own work are 10x more useful
