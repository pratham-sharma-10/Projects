Running notes on getting the most out of Claude Code.

## Things that work

- Be specific about the *outcome*, let it choose the implementation
- Keep a `CLAUDE.md` in each repo with project conventions
- Use plan mode for anything touching more than a couple of files
- Small, reviewable commits over giant ones
